## 🚀 Başlangıç / Quick Start

Projenin yerel bilgisayarınızda çalıştırılması oldukça basittir:

### Gereksinimler
*   Node.js (v18+)
*   npm (veya pnpm / yarn)

### Kurulum Adımları

1.  **Depoyu Kopyalayın (Clone):**
    ```bash
    git clone https://github.com/KULLANICI_ADINIZ/REPO_ADINIZ.git
    cd REPO_ADINIZ
    ```

2.  **Bağımlılıkları Yükleyin (Install):**
    ```bash
    npm install
    ```

3.  **Geliştirme Sunucusunu Başlatın (Dev Server):**
    ```bash
    npm run dev
    ```
    Tarayıcınızda `http://localhost:5173` adresine giderek siteyi canlı olarak görüntüleyebilirsiniz.

4.  **Üretim Sürümü Oluşturma (Production Build):**
    ```bash
    npm run build
    ```
    Bu komut, projenin en optimize halini `dist` klasörü altında derleyecektir.

## 🌐 Vercel ile Canlıya Alma / Deploy to Vercel

Sitenizi Vercel üzerinde yayınlamak için:
1.  Vercel hesabınıza giriş yapın.
2.  **Add New Project** seçeneğini seçerek GitHub deponuzu bağlayın.
3.  Projenizi seçip **Deploy** butonuna tıklayın. Vercel, Vite yapılandırmasını otomatik olarak algılayıp birkaç saniye içinde sitenizi yayınlayacaktır.
4.  Depoda yapacağınız her `git push` güncellemesinde Vercel sitenizi otomatik olarak günceller.
